# Feature Map

The Greenfield Ward Citizens' Portal (WardConnect) has the following features and page routes:

- `/login`: SMS OTP-based authentication (simulated OTP for POC).
- `/`: Resident dashboard showing active/resolved/total complaint counters, quick actions, audio-guide, and recent complaints.
- `/raise-complaint`: Multi-step form to select category, enter details, prioritize, capture photos (via device camera or webcam upload), and record voice descriptions (microphone input). Includes text-to-speech confirmation.
- `/complaints`: Historic timeline of reported grievances on the device with photo display, audio player, location, and deletion capabilities.
- `/updates`: Notices/notices panel displaying community news/sales with elderly-friendly text-to-speech audio reader.
- `/profile`: Resident profile screen allowing edits to name, ward, house number, and logout.
